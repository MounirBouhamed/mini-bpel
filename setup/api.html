• Run a BPEL process: http://localhost:8080/?bpel_file=http://filelocation/file.bpel
  • Display running process: http://localhost:8080/?id=&action=displayText
  • Display XML of running process: http://localhost:8080/?id=&action=displayXml
  • Retrieve a variable: http://localhost:8080/?id=&action=getVar&name=
  • Set a variable: http://localhost:8080/?id=&action=setVar&name=&value=
  • Pause execution(Acts as kill): http://localhost:8080/?id=&action=pause
  • Resume a paused thread: http://localhost:8080/?id=&action=resume
  • Get the bpel file for a thread: http://localhost:8080/?id=&action=getBpel
  • Get the log for a thread: http://localhost:8080/?id=&action=log
  • Get the highest thread id: http://localhost:8080/?action=getId
  • NOTE: Not giving id=# will act upon the highest thread id (The most recent)
